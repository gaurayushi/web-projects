# 🚀 Ayushi Dev Lab — Frontend & Full-Stack Project Collection

A collection of responsive frontend and full-stack projects built using **HTML, CSS, JavaScript, React, Next.js, Tailwind CSS, Node.js, Express, and MongoDB**.

This repository contains multiple projects created for learning, practice, portfolio building, and real-world UI/UX implementation.

---

## 🚀 Projects Included

### 📚 Interactive Book Landing Page

An elegant, responsive, and animated book landing page built using **HTML, CSS, and JavaScript**, with real-time book search using the **Google Books API**.

🔗 **Live Preview:**  
https://genuine-yeot-035261.netlify.app/

📁 **GitHub Folder:**  
https://github.com/gaurayushi/ayushi-dev-lab/tree/main/Bookhub

#### ✨ Features

- Book search powered by Google Books API
- Fully responsive layout for desktop, tablet, and mobile
- 3D book card stacking effect
- Smooth scroll animations
- Clean semantic HTML, CSS, and JavaScript
- Optimized for performance and accessibility

### ✅ Task Tracker App

A full-stack task and project management application built with **React, Node.js, Express, and MongoDB**.

🔗 **Frontend Live Demo:**  
https://web3-hdl3.vercel.app/

📁 **GitHub Folder:**  
https://github.com/gaurayushi/ayushi-dev-lab/tree/main/task-tracker

#### ✨ Features

- JWT-based authentication
- Login and signup
- Google authentication
- User profile with avatar
- Create and delete projects
- Create, update, and delete tasks
- Sidebar navigation
- Fully responsive UI
- Token handling with localStorage

#### 🛠️ Tech Stack

| Frontend | Backend | Database |

| React, Tailwind CSS | Node.js, Express.js | MongoDB |
