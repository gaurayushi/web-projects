# 🌐 Web3 — Frontend & Full-Stack Project Collection

A collection of responsive frontend and full-stack projects built using **HTML, CSS, JavaScript, React, Next.js, Tailwind CSS, Node.js, Express, and MongoDB**.

This repository contains multiple projects created for learning, practice, portfolio building, and real-world UI/UX implementation.

https://pokeapi-ayushigaur.vercel.app/
